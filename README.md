# GoProExtractor
GoProで撮影した動画（mp4）からGPSデータを抽出し，GPXおよびKML形式のファイルを出力するwindows GUIアプリ（C#，wpf）です． （HERO 10で撮影したmp4ファイルで動作確認しています）
